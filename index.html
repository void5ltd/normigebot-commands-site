Fun Commands

  • nb.8ball <yes/no question> | NormigeBot will answer your yes/no question
  • nb.d6 | NormigeBot rolls a six-sided dice
  • nb.d20 | NormigeBot rolls a twenty-sided dice
  • nb.coinflip | NormigeBot flips a coin

Animal-themed Commands

  • nb.doggo | Random Dog image
  • nb.catto | Random Cat image
  • nb.foxy | Random Fox image

Space-themed Commands

  • nb.spacex:nextlaunch | Gives details about the next SpaceX Launch, including webcast link, launch window, static fire, payload, etc...

Economy Commands

  • nb.earn | Claim your income, if available
  • nb.bank:openaccount | Open an account with NormCo Bank
  • nb.bank:balance | Check your NormCo Bank account balance

Other Commands

  • nb.http:<status_code> Returns the description of the specified status_code (e.g. 404, 500)

Server Admin Commands

  • nb.permgrant <permission_level> <@user> Grants @user the permission_level you specify
  • nb.permcheck <@user> Returns @user's rank in the server
